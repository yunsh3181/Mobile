파파존스 Mobile 통합 v43.8.9

변경 사항
1. 초기화면의 기존 다크그린 배경, 로고, 헤드라인, 혜택 배너 유지
2. 먹고가기/포장하기 선택 영역만 이미지형 선택 박스로 교체
3. 2인 세트 선택 시 하프앤하프 및 크러스트 단계를 생략하고 피자 선택으로 바로 이동
4. 세트 음료 선택 완료 후 장바구니로 이동
5. 장바구니에서 여러 세트/단품 추가, 수량 변경, 삭제, 합계 확인 가능
6. 크러스트 선택 화면에서 이미지를 제거하고 이름, 설명, 추가금만 표시
7. 3인/4인 세트의 통합 크러스트 선택 영역도 텍스트형으로 변경
