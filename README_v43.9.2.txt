파파존스 Mobile 통합 v43.9.2 - 좌석 이용시간 자동 제어

적용 사항
1. 파파존 / 야외존
   - 매일 11:00~21:00
   - 라스트오더 20:30
2. 보틀존 / 보틀룸존
   - 평일 11:00~14:00
   - 토요일 15:00~21:00, 주류 주문 고객만 이용 가능
   - 토요일 라스트오더 20:30
   - 일요일 휴무
3. 모든 좌석 구역 카드는 항상 표시
4. 이용 불가 구역은 회색 비활성화 표시
5. 비활성화 카드 클릭 시 이용 불가 사유와 운영시간 안내
6. 토요일 보틀존/보틀룸존 선택 시 주류 이용 조건 확인 팝업
7. 한국 표준시(Asia/Seoul)를 기준으로 상태 자동 계산

주의
- 시간 제어는 고객 화면의 선택 UI를 위한 운영 보조 기능입니다.
- 단말의 인터넷 시간이 비정상인 경우 표시 시간이 달라질 수 있습니다.
