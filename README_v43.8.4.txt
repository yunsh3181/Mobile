파파존스 Mobile 통합 v43.8.4 수정사항

1. 모바일 더블탭 확대 방지
- 빠른 연속 터치 시 Safari/모바일 브라우저 화면 확대 차단
- 핀치/제스처 확대 방지 보강

2. 단계 이동 시 스크롤 최상단
- 주문 단계가 변경될 때마다 화면을 맨 위로 이동
- 같은 단계 안에서 좌석 선택 등 상태만 변경될 때는 현재 스크롤 위치 유지

3. 페이지별 카드 자동 크기
- 선택 카드 개수에 따라 1개/2개/3~4개/5~6개/다수 레이아웃 자동 조정
- 모바일에서는 2열 중심으로 카드 높이와 간격 자동 축소

4. 좌석 선택 상태 박스 개선(좌석 페이지에만 적용)
- 미완료: 빨간색, '좌석을 선택해 주세요' 문구 확대
- 완료: 초록색, '✓ 좌석 선택 완료'로 변경
- 선택 좌석/필요 좌석 정보는 두 번째 줄에 표시
