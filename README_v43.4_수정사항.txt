파파존스 Mobile 통합 v43.4 수정사항

1. 관리자 페이지 URL 접속 보완
- /Mobile/admin.html 접속 유지
- /Mobile/admin/ 접속 시 admin.html로 자동 이동
- /Mobile/admin-login.html 보조 접속 주소 추가

2. 세트 메뉴 씬도우 제한
- 세트 선택 시 THIN 탭과 씬도우 피자를 숨김
- 이전 선택 정보에 씬도우가 남아 있어도 세트 주문에서는 자동 초기화
- 일반 피자 주문에서는 THIN 메뉴 유지

3. 모바일 화면 압축
- 모바일 메뉴 카드를 2열로 표시
- 카드 이미지, 여백, 글자, 수량 버튼 축소
- 상단 헤더 및 진행 표시 축소
- 하단 주문 버튼 높이 축소 및 고정 유지
- 좌석, 인원, 옵션, 팝업 화면도 모바일에 맞게 압축
