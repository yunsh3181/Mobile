파파존스 Mobile 통합 v43.4
============================

GitHub Mobile 저장소 최상위에 이 폴더 안의 파일과 assets 폴더를 전부 덮어쓰기 하세요.

주소
- 모바일 주문: https://yunsh3181.github.io/Mobile/
- 관리자: https://yunsh3181.github.io/Mobile/admin.html
- 좌석관리: https://yunsh3181.github.io/Mobile/seats.html

반영 내용
1. 해피아워 배너 바로 주문 연결
   - 16:00~20:00에는 바로 주문
   - 그 외 시간에는 예약 유도
   - 예약 선택 범위는 당일 16:00~20:00만 허용
2. 포장 배너 클릭 시 포장/20% 할인/L 피자선택 화면으로 바로 이동
   - 피자 화면에서 L/F 선택 가능
3. 스프라이트 제로 배너 삭제, 하이네켄 생맥주 1잔 8,000원 광고 배너 표시
4. 음료 메뉴에 생수 500ml 1,000원, 버드와이저 병맥주 5,000원 추가
5. 관리자 주문 카드의 S/D 코드를 실제 제품명으로 표시
6. 관리자 주문 카드에 상세보기/상세접기 추가

주의
- 파일 일부만 올리지 말고 전체를 덮어쓰기 해야 합니다.
- 배포 후 1~3분 기다린 뒤 시크릿 창 또는 Ctrl+F5로 확인하세요.
