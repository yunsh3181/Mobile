파파존스 모바일 키오스크 v43.6.2 변경사항

1. 세트메뉴 선택 흐름
- 세트메뉴 → 2인/3인/4인 세트 → 한 판/하프앤하프 → 피자 선택 순서로 변경
- 2인 세트(R)는 하프앤하프 선택 불가
- 3인/4인 세트는 하프앤하프 선택 가능

2. 세트 가격 계산
- 세트 기본금액은 유지
- 하프앤하프 선택 시 +1,000원만 합산
- 크러스트 추가금, 토핑 추가금, 추가 사이드/음료 금액을 별도 합산
- 피자 두 종류의 일반 판매가격은 세트 금액에 다시 합산하지 않음

3. 세트 도우
- 세트 선택 시 오리지널 도우로 고정
- 4인 세트도 오리지널 도우만 적용

4. 관리자 주문내역
- 토핑 코드(T001 등)를 실제 한글 토핑명으로 표시
- 도우 및 하프앤하프 여부가 주문내역에 표시되도록 전송 필드 보완

버전: mobile-v43.6.2
