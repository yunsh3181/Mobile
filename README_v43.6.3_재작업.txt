수정 내용
1. 세트 피자 구성 선택 시 이전 피자 선택값을 완전히 초기화합니다.
2. 하프앤하프 첫 번째 피자가 수퍼파파스로 남거나 고정되는 현상을 수정했습니다.
3. 하프앤하프 확인창의 '다시 선택하기' 버튼은 첫 번째 피자 선택 화면으로 돌아갑니다.
4. 세트 주문의 크러스트 화면에서 씬과 크루아상을 제외했습니다.
5. 세트 주문은 내부 도우값을 오리지널로 유지합니다.
6. 세트 가격은 기본 세트가 + 하프앤하프 1,000원 + 크러스트/토핑 추가금 방식 그대로 유지합니다.
