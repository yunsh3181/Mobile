파파존스 Mobile 통합 v43.6.1

변경사항
1. 세트메뉴 음료 선택 후 진행 안정화
   - 세트 포함 음료 선택 즉시 추가 음료 선택 팝업을 표시
   - 추가 주문 여부 선택 후 주문 확인 단계로 정상 진행
2. 더 좋은 혜택 비교 조건 개선
   - 같은 이용방법, 피자 구성, 사이즈를 유지하는 혜택만 비교
   - 세트메뉴와 일반 주문 간 교차 비교 제거
   - 사이즈나 포함 메뉴 구성이 달라지는 UP&UP/세트 전환 추천 제외
3. 주문번호 안내 화면 개선
   - 카운터 결제 안내를 큰 빨간 안내 박스로 표시
   - 결제 완료 후 피자 제조 시작 문구 추가
   - 처음으로 버튼 추가
