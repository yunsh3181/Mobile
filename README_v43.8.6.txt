파파존스 Mobile 통합 v43.8.6

주요 변경사항
1. 먹고가기 1인 선택 시 좌석 1개만 선택되도록 수정
   - 다른 좌석 선택 시 기존 선택 자동 해제
2. 세트메뉴 초기 카드 디자인 개선
   - 흰색 이미지 배경을 아이보리 톤으로 변경
   - 음식 및 음료 이미지 확대
   - 4인 세트를 '가장 인기 / BEST' 카드로 변경
3. 세트메뉴의 한판/하프앤하프 및 크러스트 선택을 한 화면으로 통합
4. 처음으로/이전 버튼을 주황색으로 변경
5. 하프앤하프 규칙 적용
   - 햄 머쉬룸 식스 치즈, 프리미엄 직화불고기, 스파이시 치킨랜치 제외
   - 일반 가능 피자끼리 자유 조합
   - 존스 페이버릿은 식스 치즈와만 조합
   - 식스 치즈는 존스 페이버릿과만 조합
   - 두 번째 선택 화면에는 조합 가능한 피자만 노출
6. 음료 카드 내부를 좌측 세로 이미지 / 우측 용량·수량 선택 구조로 개선
7. 이미지 잘림 완화를 위한 위치 및 표시 방식 보정

기준 버전: v43.8.4
