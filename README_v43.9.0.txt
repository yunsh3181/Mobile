파파존스 Mobile 통합 v43.9.0 안정화 수정사항

1. 초기화면을 v43.8.7 방식으로 복구
- 별도 통합 이미지(home_dine_takeout.png)를 사용하지 않음
- 기존 먹고가기/포장하기 반응형 버튼 복원
- 모바일 이미지 잘림 문제 해소

2. 주문 관리자 미접수 반복 알림
- payment_pending/new 주문이 있는 동안 10초 간격 반복
- 접수/취소 등 미접수 상태 해제 시 반복 즉시 종료
- 탭 제목에 🔴 미접수 주문(건수) 표시

3. 음성 톤
- 한국어 여성 음성을 우선 선택
- 브라우저 TTS의 음높이와 속도를 높여 여자아이에 가까운 톤으로 조정
- 새 주문: 새로운 주문이 접수되었습니다.
- 반복: 미접수 주문이 있습니다. 확인해 주세요.

주의: 브라우저/운영체제에 설치된 TTS 음성에 따라 실제 목소리는 달라질 수 있습니다. 관리자 화면에서 한 번 클릭해야 자동재생 제한이 해제될 수 있습니다.
