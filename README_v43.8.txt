파파존스 Mobile 통합 v43.8 수정사항

1. 만석 구역 줄서기
- 인원 조건에 맞는 구역은 만석이어도 선택할 수 있습니다.
- 만석 구역 카드에 "만석 · 줄서기 가능"을 표시합니다.
- 구역 진입 후 사용 중인 각 테이블에서 줄서기를 등록할 수 있습니다.

2. 모바일 초기 화면
- 먹고가기/포장하기 카드를 작은 정사각형 2열 구조로 변경했습니다.
- 모바일 한 화면에서 주요 선택 카드와 프로모션 영역이 더 잘 보이도록 축소했습니다.

3. 일반 주문 피자 사이즈 안내
- R 레귤러: 9인치 / 1~2인용
- L 라지: 12인치 / 2~3인용
- F 패밀리: 14인치 / 3~4인용
- 권장 인원 안내 문구를 추가했습니다.
