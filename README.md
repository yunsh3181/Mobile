## README.txt

파파존스 키오스크 프로토타입 v41
================================

UI/가독성 전면 개선
- 밝은 카드: 제목 검정, 설명 진회색, 가격 파파존스 레드
- 다크그린 배경: 제목 흰색, 설명 연한 회색
- 팝업·주문확인·사이드·음료 카드의 낮은 대비 문제 수정
- 유료 추가 주문 배너를 빨간 배경/흰 글씨로 통일
- 메인/완료/취소 버튼 디자인 규격 통일

초기 화면
- 먹고가기·포장하기 카드를 같은 다크그린 계열로 통일
- 빨간 대형 카드 제거
- Hover/Glow/Scale 효과 적용
- 포장 박스 이모지 제거 및 라인 아이콘 적용

아이콘
- 기능성 이모지를 통일된 SVG 라인 아이콘으로 교체
- 먹고가기, 포장, 좌석, 룸, 야외, 혜택, 세트, 할인, 시간,
  사이드, 음료, 전화번호, 장바구니, 결제, 완료 등에 적용
- 음식은 실제 제품 사진 유지

세트 메뉴
- 피자·파스타·치킨·콜라를 실제 제품 사진으로 표시

애니메이션/피드백
- 페이지 Fade/Slide
- 팝업 Zoom
- 카드 Hover/선택 Glow
- 선택 시 짧은 UI 클릭음
- 메뉴 이미지 미리 불러오기

진행률
- 주문방식 → 혜택 → 피자 → 사이드 → 음료 → 결제
- 완료 초록, 현재 빨강, 예정 회색
- 모바일에서도 가로 스크롤 가능한 컴팩트 구성

추천
- 사이드·제로 음료에 간단한 추천 배너 추가
- 우측 고정 주문요약은 모바일 공간 문제로 제외


검증
- 기존 v40 혜택 도우미 모달의 닫힘 오류 수정
- shell/render 화면 렌더링 함수 복구
- JavaScript 문법 검사 통과

파파존스 모바일 키오스크 v43.2.1 Hotfix
======================================

수정 내용
- GitHub Pages에서 data.js 경로 또는 캐시 문제로 빈 화면이 발생하는 현상 수정
- 메뉴 데이터(data.js)를 index.html 안에 직접 포함
- Firebase 초기화 설정도 index.html 안에 직접 포함
- 앱 시작 중 오류가 발생하면 초록색 빈 화면 대신 오류 안내 화면 표시
- Firebase 주문 저장 기능은 v43.2와 동일하게 유지

GitHub 업로드 방법
1. 이 폴더 안의 모든 파일과 assets 폴더를 Mobile 저장소의 최상위에 업로드
2. index.html이 반드시 저장소 최상위에 있어야 함
3. GitHub Pages 설정에서 배포 브랜치의 /(root) 선택
4. 업로드 후 Ctrl+F5 또는 시크릿 창에서 확인

주의
- 폴더 전체를 한 단계 더 감싸서 업로드하지 마세요.
- URL이 /Mobile/ 이라면 /Mobile/index.html 위치에 파일이 있어야 합니다.

## README_FIREBASE.txt

PAPA JOHNS 판교2테크노밸리점 v22 — Firebase 실시간 주문 데모

1. GitHub Pages 업로드
- 이 압축파일 내부 파일 전체를 papajohns-kiosk2 저장소의 루트에 덮어씁니다.
- 키오스크: https://yunsh3181.github.io/papajohns-kiosk2/?v=22
- 관리자: https://yunsh3181.github.io/papajohns-kiosk2/admin.html?v=22

2. Firestore 규칙
- Firebase Console > Firestore Database > 규칙
- FIRESTORE_RULES_TEST.txt 안의 규칙을 붙여넣고 게시
- 테스트 모드가 아직 유효하면 바로 동작할 수 있지만, 규칙을 명시적으로 확인하는 것을 권장합니다.

3. 테스트
- 관리자 페이지를 먼저 열고 '알림음 켜기'를 한 번 누릅니다.
- 다른 휴대폰/브라우저에서 키오스크 주문을 완료합니다.
- 관리자 페이지에 주문 카드가 실시간 표시되고 알림음이 납니다.
- 접수 > 조리 시작 > 조리 완료 상태 변경을 테스트합니다.

주의
- 현재 주문 완료 버튼은 실제 카드 결제 없이 Firestore에 주문을 저장합니다.
- 테스트용 공개 규칙이므로 실제 운영 전 Firebase Authentication을 연결해야 합니다.
